# Tata GenAI Powered Data Analytics Job Simulation

## Overview

This repository contains my completed work for the Tata Group GenAI Powered Data Analytics Job Simulation hosted by Forage.

The project focused on applying data analytics, predictive modeling concepts, responsible AI principles, and business strategy to solve customer delinquency challenges in a financial services environment.

---

## Project Tasks

### Task 1: Exploratory Data Analysis (EDA)

* Assessed dataset quality and completeness
* Identified missing values and anomalies
* Highlighted key delinquency risk indicators

### Task 2: Predictive Model Planning

* Selected Logistic Regression as the primary model
* Identified important predictive variables
* Defined evaluation metrics and fairness checks

### Task 3: Business Recommendations

* Developed stakeholder-focused recommendations
* Created SMART intervention strategies
* Incorporated responsible AI considerations

### Task 4: AI-Powered Collections Strategy

* Designed a scalable AI collections framework
* Defined agentic AI responsibilities
* Proposed fairness, explainability, and compliance guardrails
* Evaluated expected business outcomes
